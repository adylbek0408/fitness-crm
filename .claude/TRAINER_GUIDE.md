# Инструкция для тренера / администратора

Документ написан для тех, кто будет каждый день пользоваться модулем
обучения CRM Asylzada. Технических знаний не требуется.

> Для всех действий нужен логин **администратора или регистратора**
> на `https://crm.aiym-syry.kg/login`. Студенты заходят на отдельный URL
> `https://crm.aiym-syry.kg/cabinet`.

---

## Часть 1. Видеоурок: загрузить и опубликовать

1. Заходим на сайт CRM, авторизуемся.
2. В левом меню → **«Уроки»**.
3. Нажимаем кнопку **«Загрузить новый урок»**.
4. Заполняем:
   - **Название** — короткое, как на тумблере (например, «Урок 5 — приседания»).
   - **Тип** — Видео.
   - **Описание** — что будет в уроке (1-2 строки).
   - **Группы** — кому будет доступен. Зажимаем `Ctrl` (или `Cmd` на Mac)
     чтобы выбрать несколько.
   - **Файл** — MP4 файл с компьютера. Размер до 300 МБ
     (если больше — пишем разработчику, добавим поддержку TUS).
5. Жмём **«Загрузить и опубликовать»**.
6. Ждём, пока полоска прогресса дойдёт до 100%.
7. **Cloudflare** ещё 1-2 минуты обрабатывает видео — длительность и превью
   подтянутся сами. Можно сразу обновить страницу — урок уже в списке.

### Что увидит студент

- Зайдёт в кабинет → блок **«Мои уроки»** → увидит ваш урок.
- Откроет → начнёт смотреть. На видео по диагонали будет идти его имя
  и телефон (защита от перезаливки).

### Если что-то пошло не так

| Проблема | Что делать |
| --- | --- |
| 503 Service Unavailable при загрузке | В `.env` пуст `CF_STREAM_API_TOKEN` |
| Видео не загружается на 100% | Файл >300 МБ или плохой интернет — попробовать снова |
| После загрузки не видно превью | Подождать 2-3 минуты, обновить |
| Студент пишет «не вижу урок» | Проверьте, добавлена ли его группа в урок |

## Часть 2. Аудиоурок

Те же шаги, что и видео, но:
- **Тип** = Аудио.
- Файл MP3 или WAV.
- Загружается обычно за 10-30 секунд (R2 быстрее CF Stream).

В кабинете студент увидит карточку с иконкой наушников вместо
видео-превью.

## Часть 3. Прямой эфир

> **Важно:** для эфира нужен компьютер с OBS (бесплатно — `obsproject.com`).
> Телефон не подойдёт.

### Один раз — настроить OBS

1. Скачать **OBS Studio** с `obsproject.com` → установить.
2. Открыть OBS, в правом нижнем углу нажать **«Настройки»**.
3. Слева → **«Вещание»**.
4. Поле **«Сервис»** → выбрать **«Custom»** (Настраиваемый).
5. Здесь нужны два значения, которые мы получим в CRM на следующем шаге.
6. Внизу нажимаем **OK** (закроется без сохранения, ничего страшного).

### Каждый раз перед эфиром

1. В CRM → **«Эфиры»** → **«Создать новый эфир»**.
2. Заполнить:
   - **Название** (например, «Лекция 1 — введение»).
   - **Описание** — 1 строка.
   - **Группы** — кто увидит эфир.
3. Нажать **«Создать эфир»** — появится карточка.
4. В карточке два поля:
   - **RTMP URL** — это **«Сервер»** в OBS.
   - **Stream Key** — это **«Stream key»** (ключ потока) в OBS.
5. Открываем OBS → Настройки → Вещание → вставляем оба.
6. **OK**, выходим из настроек.

### Запуск эфира

1. В OBS → выбираем сцену с камерой и микрофоном
   (если нет — Источники → + → «Источник видео» / «Захват аудио»).
2. **«Начать трансляцию»** в OBS.
3. **Через 30-60 секунд** в CRM нажимаем **«Старт»** на карточке эфира.
4. Эфир пошёл. Студенты в своём кабинете → **«Прямой эфир»** уже видят.

### Завершение

1. В CRM → **«Завершить»** на карточке эфира.
2. В OBS → **«Остановить трансляцию»**.
3. **Через 1-3 минуты** Cloudflare сделает запись и в CRM/кабинете она
   появится в **«Архив эфиров»** автоматически.

### Студенты

- Видят список зрителей друг друга, как в Zoom.
- Если поставят на паузу или свернут вкладку — heartbeat пропадает,
  через 30 секунд их имя уходит из списка.

## Часть 4. Онлайн консультация (1-на-1)

Это **не запись на консультацию**, а **готовая ссылка-комната**, которую
тренер отправляет в WhatsApp клиенту.

### Создать ссылку

1. CRM → **«Консультации»** → форма.
2. Заполнить:
   - **Название** — для себя, чтобы не путать (например, «Анна —
     консультация по питанию»).
   - **Тренер** — кто проводит.
   - **Клиент** — для кого (или оставить «открытая ссылка», если
     клиент не зарегистрирован).
   - **Срок (часы)** — через сколько ссылка перестанет работать. По умолчанию 24.
   - **Макс. заходов** — сколько раз можно зайти. По умолчанию 2
     (тренер + клиент).
3. **«Создать ссылку»**.
4. Появилась карточка → нажать **«WhatsApp»** → откроется WA с готовым
   сообщением → выбрать клиента → отправить.

### Проведение

1. Тренер заходит по той же ссылке.
2. Клиент заходит по ссылке из WhatsApp.
3. Оба попадают в одну Jitsi-комнату.
4. По умолчанию доступна камера, микрофон, расшаривание экрана, чат.

### Важно

- Ссылка **одноразовая в плане доступа**: 2 захода и всё.
- После завершения — нажать **«Отменить»** на карточке (на всякий случай).
- Если клиент потерял ссылку — создать новую.

## Часть 5. Аналитика — кто что смотрит

CRM → **«Аналитика»**.

- **Всего уроков** — сколько у вас опубликовано.
- **Студентов с доступом** — сколько клиентов имеют группу с привязанными уроками.
- **Среднее %** — насколько в среднем досматривают уроки.
- **Неактивны 7+ дней** — кто не открывал уроки больше недели.

Полезные сценарии:
- В фильтре выбрать группу → увидеть, как именно эта группа учится.
- Кликнуть на урок → откроется список зрителей с прогрессом каждого.
- Изменить «Неактивны с (дней)» на 14 → увидеть тех, кто давно не заходил.

## Часто задаваемые вопросы

**Q: Студент не видит урок.**
A: Проверьте, что его группа добавлена в урок (Уроки → редактировать).

**Q: Видео тормозит у студента.**
A: Cloudflare Stream сам подбирает качество по интернету. Если у студента
3G — будет 360p, на WiFi — 1080p. Тормоза обычно от его сети.

**Q: Можно ли скачать урок на телефон, чтобы смотреть оффлайн?**
A: Нет. Это сделано специально (защита от воровства).

**Q: Студент сделал скриншот видео.**
A: На скриншоте будет его имя и телефон поверх картинки. Если он этим
поделится — мы сразу узнаем, кто слил.

**Q: Сколько клиентов одновременно могут смотреть эфир?**
A: Cloudflare Stream держит тысячи зрителей, проблем не будет.

**Q: Можно ли проводить эфир с телефона?**
A: Можно, но через OBS Mobile (есть на iOS/Android), сложнее настроить.
Проще — с компьютера.

**Q: Сколько хранится запись эфира?**
A: Бессрочно, пока вы не удалите урок из админки.

**Q: Что если интернет упал во время эфира?**
A: OBS попробует переподключиться сам. Если не получится — нажать
«Завершить» в CRM, остановить OBS, создать новый эфир.
