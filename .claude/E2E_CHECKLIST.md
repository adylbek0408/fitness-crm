# E2E чек-лист — модуль обучения

Прогон **полностью вручную** перед боевым релизом и после каждого
крупного апдейта `apps/education` или `frontend-spa/src/pages/{admin,cabinet}/education`.

Чтобы пройти чек-лист, нужны:
- Заполненный `.env` (см. `.env.example` секция «Education»).
- Cloudflare R2 бакет с custom domain (для аудио).
- Cloudflare Stream API токен и signing key.
- Зарегистрированный webhook (`.claude/WEBHOOK_SETUP.md`).
- Jitsi self-host доступен (`JITSI_DOMAIN`).
- Тестовый клиент в БД с привязанной группой.

> Каждый шаг — `[ ]` пока не проверен, `[x]` после прохождения. Если упало —
> зафиксировать в `.claude/HANDOFF.md` и не закрывать пункт.
